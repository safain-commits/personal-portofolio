# personal-portofolio

Portfolio web app with:
- React + Vite frontend
- Express backend API
- MySQL database
- Admin CMS for project content, media, and 3D viewer settings

## Structure
- `frontend/` — public site and admin UI
- `backend/` — API, auth, uploads, database access
- `docs/` — changelogs and supporting documentation
